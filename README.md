<h4 align="center">基于SpringBoot+Vue3前后端分离的个人网站</h4>

## 前端运行

```bash
# 安装依赖
yarn --registry=https://registry.npmmirror.com

# 启动服务
yarn dev

# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:80
```
